# 歌单迁移工具 🎵

一键跨平台迁移音乐歌单 — 截图识别 → 扫码登录 → 批量导入

## 这是什么

不同音乐 App 的版权不一样，你在酷狗收藏的歌，网易云可能听不了。换个 App 就得重新一首首搜歌，太累了。

这个工具帮你搞定：**手机长截图歌单 → 自动识别 → 一键导入到指定平台**。整个过程在你自己电脑上运行，数据不会上传到任何第三方。

## 目前支持

| 平台 | 作为目标 (导入) | 状态 |
|------|:---:|------|
| 网易云音乐 | ✅ | 已支持 |
| QQ音乐 | ⏳ | 开发中 |
| 酷狗音乐 | ⏳ | 开发中 |
| 酷我音乐 | ⏳ | 开发中 |

截图的来源不受限制，任何 App 的截图都能识别。

## 怎么用

### 第一次使用

1. 电脑上双击 `install.bat`（等几分钟装依赖）
2. 双击 `start.bat` 启动
3. 手机连同一个 WiFi，浏览器打开显示的网址

### 以后每次用

双击 `start.bat` → 手机打开网址 → 搞定

### 操作步骤

1. **截图** — 在源 App 对歌单页面长截图（1-2 张）
2. **上传** — 手机浏览器打开工具，上传截图
3. **核对** — OCR 自动识别歌名，可手动编辑修正
4. **登录** — 选择目标平台，扫码登录
5. **导入** — 自动搜索匹配，标记已收藏，一键导入新歌

## 前置要求

- 电脑安装 [Node.js](https://nodejs.org/) (18 或以上)
- 手机和电脑在同一 WiFi 下

## 开发

```bash
# 安装依赖
npm install
cd server && npm install
cd ../client && npm install

# 开发模式（前后端热更新）
cd ..
npm run dev

# 生产模式
npm start
```

## 添加新的音乐平台

1. 在 `server/src/adapters/` 下新建文件夹
2. 实现 `IPlatformAdapter` 接口（参考 `netease/netease.adapter.ts`）
3. 在 `server/src/index.ts` 中注册
